Topics Insights API Documentation

Created by Amit Rechavia & Maor Elfassy

Endpoint: /signUp

Sign up a new user with the given topics.

Example: /signUp?user_id=my_user_id&topics=topic1,topic2,topic3

Endpoint: /analyze

Analyze the excitement level of people according to the tweets for a given topic.

Example: /analyze?topic=topic_name

Endpoint: /visualize

Visualize the level of excitement for the tweets of a given topic.

Example: /visualize?topic=topic_name

If you want to see the graph in your machine, save the returned values as a variable named 'data' and run the following code:


import pandas as pd
import matplotlib.pyplot as plt

df = pd.DataFrame(json.loads(data))
plt.plot(df['X'], df['Y'])
plt.xlabel('tweet number')
plt.ylabel('insight satisfaction')
plt.title('tweet to satisfaction graph')
plt.grid(True)
plt.show()
